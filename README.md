# One-Piece: Grand Line Adventures

## Bem vindo a One Piece: Grand Line Adventures 🍇!

One Piece: Grand Line Adventures é um jogo criado por mim como trabalho para a 
cadeira de Programação Orientada a Objetos da Universidade do Algarve.

Este jogo contém as seguintes funcionalidades:

- 🦜 Criar um tripulante (Nome, Classe, Fruta, Recompensa, Poder, Energia)
- ⛵ Criar um Navio 
- ➕ Recrutar Tripulantes
- 🦈 Expulsar Tripulantes
- 🔢 Calcular o poder total do seu e de outros navios
- 📋 Exibir um manifesto da sua tripulação


## English
## Welcome to One Piece: Grand Line Adventures 🍇!

One Piece: Grand Line Adventures is a game I created as a project for my Object-Oriented Programming course at the University of Algarve.

This game contains the following features:

- 🦜 Create a crew member (Name, Class, Fruit, Reward, Power, Energy)
- ⛵ Create a Ship
- ➕ Recruit Crew Members
- 🦈 Expel Crew Members
- 🔢 Calculate the total power of your ship and other ships
- 📋 Display your crew's manifest
